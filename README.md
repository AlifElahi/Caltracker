# Caltracker

Private calorie calendar app with login, manual food entries, walking/workout burn, weight logging, session reports, and JSON file storage.

## Run locally

```bash
npm run dev
```

Open `http://127.0.0.1:5173`.

## Render

Use:

- Build command: `npm install`
- Start command: `npm start`

The app stores data in local JSON files under `data/`. On hosted platforms, make sure the filesystem is persistent if you need data to survive redeploys/restarts.
